# Opret nyt Javalin web projekt i IntelliJ

Dette er en Github repository template, som du kan bruge, når du skal i gang med et nyt Javalin projekt, og ikke vil bruge tid på basisopsætning af projektet. 
Du får automatisk genereret et nyt repository med samme directory struktur og filer som skabelonen.

## Sådan gør du

Du skal oprette dit eget repository baseret på denne template og give projektet dit eget navn.

### Trin

1. Klik på den grønne **"Use this template"** knap.

2. Vælg **"Create a new repository"**.

3. Giv dit projekt et navn.
   
4. Klik **Create repository**.

GitHub opretter nu et **nyt repository med samme struktur som templaten**, men med:

- dit eget repository-navn  
- en ny og tom Git-historik  
- dine egne commits

### Klon dit repository

Når repository er oprettet, skal du klone **dit eget repository** (ikke templaten):

```bash
git clone https://github.com/<brugernavn>/<repository>.git
```
